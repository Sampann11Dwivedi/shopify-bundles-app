# shopify-bundles-app
Shopify app that enables merchants to create and manage checkout-based product bundles and upsells. Built using Shopify Functions and Metafields, it dynamically applies add-on products at discounted prices during checkout using cart transform logic, enhancing average order value and providing a seamless shopping experience.

Key Points :
1. Works at checkout, not just in the cart.
2. Uses Shopify Functions for backend logic.
3. Stores configuration in Metafields.
4. Dynamically applies add-on products and discounts.
5. Focused on boosting merchant sales through bundles/upsells.
